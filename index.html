Our Portfolio

REWARDS & RECOGNITION PORTAL

Our project entails developing a Teams Extension for a top rewards and recognition portal in the US and Europe. This portal lets employees share recognition posts and redeem rewards via its marketplace. Our custom Teams App seamlessly integrates their web portal with Microsoft Teams, enabling users to perform all actions within Teams. Key features include creating recognition posts, displaying them on the Teams Activity wall, engaging with posts, and real-time notifications for user recognition.

VIEW PROJECT

401k and 941 Tax Filing Automation

Our project develops a tax filing and 401k automation system. Using Finch API aggregator, a secure middleware fetches data, with PDFTron populating tax forms. Elastic Search enables record search and dashboard creation for real-time analysis, all on a serverless architecture.

VIEW PROJECT

Compliance Assurance System

The Compliance Assurance System utilizes a microservices-based architecture in pharmaceutical manufacturing. Its primary goal is to detect compliance violations like uniform and safety gear adherence. Using TensorFlow's object detection model, it processes live CCTV feeds, addressing challenges of stream reluctance and limited edge device capacity.

VIEW PROJECT

AI Injury Proneness Detection System

The Athletic Injury Proneness Detection system utilizes AI pose estimation to assess injury risk. It records training actions, evaluating proneness via the LESS score. This AI-driven solution reduces coaching efforts, enhancing accuracy. The patented project bears the client's name.

VIEW PROJECT

Kiosk Food Ordering System

The Kiosk Food Ordering System is a native application tailored for water park kiosks, enhancing food ordering efficiency. It offers a variety of tickets and combos to users, while administrators have access to sales data for informed decision-making on popular items.
