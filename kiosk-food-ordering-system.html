Overview

The Kiosk Food Ordering System is a native-based application designed specifically for kiosk machines in a water park environment. The system streamlines the food ordering process for park visitors, enhancing their overall experience. Additionally, it incorporates a wallet system for seamless transactions using RFID bands, minimizing the need for traditional payment methods. To ensure uninterrupted service, the system features database synchronization to prevent glitches during network failures on the park premises.

challenges and solution

Developing a user-friendly application tailored for kiosk machines.

  • Built a native-based application optimized for kiosk interfaces with more system control like print etc. ensuring a seamless and intuitive food ordering experience for park visitors.

Eradicate cash/card transactions on premises

  • Built a secure custom wallet system for the application, allowing visitors to load funds onto RFID bands using secure authentication techniques for convenient and cashless transactions at food kiosks within the water park.

Ensuring load balancing on AI Engine

  • Orchestrated the AI engine deployment using Kubernetes on an EKS cluster, ensuring scalability, resource optimization, and efficient management of the object detection model.

Efficiently storing and retrieving frames for AI processing.

  • Utilized Amazon S3 for secure and scalable storage, providing a centralized repository for frames that facilitated efficient retrieval and processing by the AI engine.
